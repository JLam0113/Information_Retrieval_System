This assignment was done in Java and Eclipse IDE, you can find the class files in bin and the java files in src.
UserInterface.java is the main file used to run the application. Interaction is done using the console. 
For stopwords, the words in stopwords.txt, numbers, single letters, all special characters except hyphens and apostrophes were removed.
It will ask you for yes/no on stopwords and stemming, then ask for a query until "ZZEND" is entered.
MAP value is outputted to map.txt
R-Precision is outputted to rprecision.txt
